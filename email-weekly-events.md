## Beehiiv Setup — Step by Step

### 1. Create the newsletter post template

Beehiiv's weekly broadcast is called a **Post**, not an automation. Here's how to set up the recurring send:

1. Go to **Posts → New Post**
2. In the top-right, click **Schedule**
3. Set the send time: **Wednesday, 9:00 AM PT**
4. Each week, duplicate the previous week's post and update the event listings — takes about 10 minutes

> 💡 Beehiiv doesn't have a true "recurring template" yet, but duplicating a post is the fastest workflow.

### 2. Set your audience

- Audience: **All active subscribers** (default)
- As your list grows, you can segment by city (Irvine, Newport, Fullerton etc.) if you collect that on signup

### 3. Recommended send settings

| Setting | Value |
|---|---|
| From name | whatwedonowmama |
| Reply-to | hello@whatwedonowmama.com |
| Send day | Wednesday |
| Send time | 9:00 AM PT |
| Audience | All active subscribers |
| Web version | On (so you can share the link on social) |

### 4. Weekly workflow (every Tuesday evening)

1. Duplicate last week's post
2. Update subject line with this week's date range
3. Replace the 5 event listings with new ones
4. Update the "featured pick" at the top
5. Schedule for Wednesday 9am → hit Confirm

## Filling it in — weekly checklist

Each Tuesday, work through this:

- [ ] Pick 5 events happening Friday–Sunday
- [ ] Choose 1 as the "featured pick" — ideally free or low-cost, broad age range
- [ ] Write 2–3 sentences for the featured pick only (the rest are one-liners)
- [ ] Emoji match the vibe: 🎡 fun/rides, 🌿 outdoors, 🎨 arts/craft, 🏖️ beach, 🍎 markets, 🎭 shows, 🏃 sports, 🔬 science/museum
- [ ] Update subject line with this weekend's dates
- [ ] Schedule for Wednesday 9am PT
- [ ] Share the web version link on Instagram Stories after it sends

## Subject line formula for each week

`[emoji] [what's happening] — [Month Day–Day]`

Examples:
- `🌴 best OC weekend picks — Mar 14–16`
- `🎡 things to do this weekend — Mar 21–23`
- `🌸 spring break events in OC — Mar 28–30`

Keep it under 50 characters. The date range tells subscribers it's fresh and timely.

## Performance benchmarks to aim for

| Metric | Target |
|---|---|
| Open rate | 35–50% (newsletters with local/timely content perform well) |
| Click rate | 8–15% |
| Unsubscribe rate | < 0.3% per send |

If open rate drops below 25%, try: changing send time to Tuesday evening, testing a more curiosity-driven subject line, or shortening the email to just 3 events.
